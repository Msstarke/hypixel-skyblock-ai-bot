Place a TTF font file here for NanoVG custom text rendering.

Supported filenames (checked in order):
  - inter.ttf     (recommended: https://fonts.google.com/specimen/Inter)
  - regular.ttf
  - font.ttf

If no font is found, text falls back to Minecraft's built-in TextRenderer.
Shapes (rounded rects, shadows, arcs, gradients) still use NanoVG regardless.
