# Hypixel Skyblock Mining Guide

## Best Mining Armor (ranked)

Item IDs for price lookups: ARMOR_OF_DIVAN_HELMET, ARMOR_OF_DIVAN_CHESTPLATE, ARMOR_OF_DIVAN_LEGGINGS, ARMOR_OF_DIVAN_BOOTS, GLACITE_HELMET, GLACITE_CHESTPLATE, GLACITE_LEGGINGS, GLACITE_BOOTS, MINERAL_HELMET, MINERAL_CHESTPLATE, MINERAL_LEGGINGS, MINERAL_BOOTS

### End Game
- **Armor of Divan** (LEGENDARY) — Best in slot for mining. +150 Mining Fortune, +1,800 Mining Speed, +200 Defense, 5 gemstone slots per piece. Requires HotM 7. Full set costs ~400-600M coins on AH.

### Mid Game
- **Glacite Armor** (EPIC) — Good for Glacite Tunnels. +100 Mining Speed, Cold Resistance, Mining Fortune bonuses. Full set costs ~5-20M coins on AH.
- **Mineral Armor** (EPIC) — Good for Mining XP farming. Grants Mining Spread. Full set ~1-5M coins.

### Early Game
- **Miner Armor** (RARE) — Basic mining set with Defense bonuses inside mines. Very cheap, under 1M.
- **Prospecting Armor** (UNCOMMON) — Starter mining set. Under 100k.

## Best Drills / Pickaxes (ranked)

### End Game
- **Divan's Drill** (MYTHIC) — Best in slot mining tool. Stats: +75 Damage, +150 Mining Fortune, +10 Breaking Power, +1,800 Mining Speed. Requires HotM 7 to forge. Takes 143 days to forge all components (100 days with Quick Forge 20).

### Mid Game
- **Titanium Drill DR-X655** — Best titanium drill. Strong Mining Speed and Fortune.
- **Titanium Drill DR-X555 / DR-X455 / DR-X355** — Progressively weaker versions. Used to forge higher tiers.
- **Pickonimbus 2000** — Good all-around pickaxe, easier to obtain than Divan's Drill.

### Early Game
- **Refined Mithril Pickaxe** — Good starter mining tool in Dwarven Mines.
- **Mithril Pickaxe** — Early Dwarven Mines pickaxe.
- **Gemstone Drills** (Ruby/Topaz/Jasper) — Solid mid-game options.

## Best Mining Pets (ranked)

### For Mining Speed / Fortune (general mining)
- **Mithril Golem Pet** (LEGENDARY) — Best general mining pet. Grants Mining Speed, Mining Fortune, and increases powder gained. Highly recommended.
- **Scatha Pet** (EPIC/LEGENDARY) — Grants a chance to find worms while mining, bonus powder. Good for Crystal Hollows.
- **Bal Pet** (EPIC/LEGENDARY) — Best for Glacite Tunnels / lava fishing adjacent mining areas.

### For Mining XP
- **Silverfish Pet** (LEGENDARY) — Best pet for Mining XP gain. Provides Mining Wisdom and XP boosts.

### Early / Mid Game
- **Rock Pet** — Budget option, small Mining Speed bonus.
- **Armadillo Pet** — Defense while mining.

## Heart of the Mountain (HotM)

HotM is the mining skill tree unlocked in Dwarven Mines. Levels 1-10.
- XP is earned from completing Dwarven Mines Commissions (King Yolkar NPC).
- **Commission XP**: Tier 1 = 2,500 XP | Tier 2 = 5,000 XP | Tier 3 = 25,000 XP

### Key HotM Perks
- **Mining Speed Boost** — Active ability, huge temporary Mining Speed increase.
- **Efficient Miner** — Mines surrounding blocks automatically.
- **Quick Forge** — Reduces forge time (up to 30% at max).
- **Star Powder** — Chance to get double powder.
- **Mining Madness** — Bonus Mining Speed and Fortune.
- **Sky Mall** — Daily random buff.

### Key HotM Unlocks
- HotM 5 → Crystal Hollows unlocked
- HotM 7 → Armor of Divan equippable + can forge Divan's Drill

## Recommended Mining Locations

- **Dwarven Mines** — Best for Mithril/Titanium grinding, commissions, HotM XP
- **Crystal Hollows** — Best for gemstones, Scatha, powder
- **Glacite Tunnels** — Best for Glacite ore, unique mobs, cold-resistant gear testing
- **Deep Caverns** — Early game mining location

## Best Reforges for Mining Armor

Item IDs for price lookups: JADERALD, GLACITE_SHARD

- **Jaded** (Jaderald stone, ~3.5M) — Best reforge for Mining Fortune. +20 Mining Fortune per piece at Legendary. Use this on Armor of Divan. Stone ID: JADERALD
- **Glaciated** (Glacite Shard stone, ~60k) — Budget mining reforge. +15 Defense, +5 Mining Fortune per piece at Legendary. Use on Glacite Armor or budget builds. Stone ID: GLACITE_SHARD

**Recommendation by armor:**
- Armor of Divan → **Jaded** (best Mining Fortune, ~3.5M Jaderald stone)
- Glacite Armor → **Glaciated** (budget option, ~60k Glacite Shard stone)

## Powder

Two types of powder used to unlock HotM perks:
- **Mithril Powder** — From mining Mithril in Dwarven Mines
- **Gemstone Powder** — From mining Gemstones in Crystal Hollows

## Drill Upgrades (Drill Anvil)

Drills can be upgraded at the Drill Anvil with:
- **Fuel Tank** — Increases max fuel
- **Drill Engine** — Boosts Mining Speed
- **Upgrade Module** — Various bonuses depending on module type
- **Gemstones** — Socketed into drill slots for stat bonuses
