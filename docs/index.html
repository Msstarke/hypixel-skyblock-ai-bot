FAQ

Is this bannable on Hypixel?

SkyAI only reads your chat input and displays an overlay. It does not automate gameplay, modify packets, or interact with the server. It functions similarly to other allowed information mods.

What Minecraft version does it support?

Minecraft 1.21.10 with Fabric Loader. You need Fabric API installed.

Can I share my key?

No. Each key locks to your Minecraft account on first use. Using it on a different account will be rejected.

What if I get a wrong answer?

Press N or type !wrong. The bot automatically searches the wiki and caches the correct answer for next time.

Do I need a key to use the mod?

No. The free tier works instantly — just install and play. Upgrade when you want more questions per hour.

Can I cancel anytime?

Yes. Your key works until it expires. No auto-renewal unless you buy again.
